Media count plugin display total no of files in your WP Media Library. You can see the table format and it will show the total number of files count

== Description ==

Media Count plugin count total number of media library items (such as images, videos, documents and audio files) that have been uploaded in to the media library. 

You can see the media count by going to the WordPress dashboard and clicking on the "Media Count" menu item. This will open the media library, which displays all of the media items that have been uploaded to your site. 

The media count is displayed all the file format uploaded the list of media items. 

We can see any images format, audio, video files total count in media library. Currently there are 28 file types wil be showing the total no.of files count.
